## Sobre

O projeto Doces Sonhos foi desenvolvido com fins acadêmicos, com foco na prática de construção de interfaces modernas e organização de layouts.

A aplicação simula o site institucional de uma confeitaria, destacando a apresentação visual dos produtos, identidade visual da marca e experiência do usuário.

<br>

## Objetivo

Este projeto teve como objetivo consolidar conhecimentos em:

- Estruturação semântica com HTML5
- Estilização de páginas com CSS3
- Organização de layout (header, section e footer)
- Boas práticas de organização de código


<br>

## Tecnologias utilizadas

- HTML5
- CSS3
- IntelliJ IDEA
- Git
- GitHub

<br>

## Status do Projeto

Este projeto foi desenvolvido exclusivamente para fins acadêmicos e, até o momento, não possui integração com back-end ou banco de dados.

Você pode visualizar o projeto online clicando abaixo:

🔗 [Acessar](https://rayssaolliveira.github.io/Front-end/)

<br>

Caso queira visualizar o projeto localmente, siga os passos abaixo:

1. Clone este repositório:

```bash
git clone https://github.com/rayssaolliveira/doces-sonhos.git
```

2. Acesse a pasta do projeto:

```bash
cd doces-sonhos
```

3. Abra o arquivo `index.html` no navegador  
ou utilize a extensão Live Server no VS Code para executar o projeto.
